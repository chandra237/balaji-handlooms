# 🛍️ BalajiHandlooms – E-commerce Web Application

BalajiHandlooms is a full-stack e-commerce platform designed for showcasing and selling handloom sarees. This project is inspired by a real-world business use case and focuses on building a scalable and production-ready system.

---

## 🚀 Features

* 🔐 User Authentication using JWT
* 🛒 Cart Management System
* 📦 Order Management (In Progress)
* 👤 Role-Based Access Control (User/Admin)
* ⚡ RESTful APIs for seamless backend communication
* 🎨 Responsive UI built with React.js and Tailwind CSS
* ⚡ Caching using Redis (Planned)

---

## 🧱 Tech Stack

### Backend

### Frontend

* React.js

## 🏗️ Architecture

The backend follows a layered architecture:

Controller → Service → Repository → Database

* Clean separation of concerns
* Scalable and maintainable design
